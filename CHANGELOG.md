# Changelog

## 0.1.0 - Initial Release

- Train Deployer: place on rails, save templates from parked trains, deploy via circuit signals
- Train Monitor: auto-discovers train types by layout, outputs counts and utilization %
- Template system: saves layout, schedule, orientation, and fuel configuration
- Configurable depot station detection for utilization calculation
- Custom signal icons (locomotive with number overlay)
- Factorio alerts for missing items
- Circuit network output of deployer chest contents
